# Full Stack Developer Capstone

## Repository Name
xrwm-fullstack_developer_capstone

## Project Name
fullstack_developer_capstone

## Project Description
This project is a Full Stack Developer Capstone application developed as part of the XRWM program. It demonstrates the implementation of a scalable web-based car dealership system using modern full-stack technologies.

The application includes:
- A Django backend for handling APIs and business logic
- A React frontend for interactive UI
- REST APIs for dealers, reviews, authentication, and sentiment analysis
- Integration of user login, logout, and registration features
- Deployment-ready architecture with CI/CD support

## Features
- User Authentication (Login, Logout, Register)
- Dealer Listings
- Dealer Reviews and Ratings
- Sentiment Analysis for Reviews
- Responsive Frontend Design
- REST API Integration

## Technologies Used
- Backend: Django, Python
- Frontend: React.js
- Database: SQLite / Cloud DB
- API Communication: REST APIs
- Deployment: Cloud Platform (App Engine / similar)
